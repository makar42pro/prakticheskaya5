# prakticheskaya5
Проект реализован с использованием принципов модульного программирования и разделен на три основных функциональных модуля.
Используемые технологии
Язык программирования: Python 3.x
Среда: Консольное приложение (CLI)
Хранение данных: Временные структуры данных Python (списки и словари), имитирующие базу данных (данные не сохраняются при выходе из программы).
Структура модулей
Проект состоит из следующих файлов, соответствующих техническому заданию:

data_store.py	—	Центральное хранилище данных (имитация БД).
auth.py Аутентификация	Регистрация, вход и выход пользователей.
tasks.py Управление задачами	CRUD-операции, фильтрация и изменение статуса задач.
reports.py Отчетность	Генерация статистики по задачам пользователя.
main.py	—	Главный файл приложения, объединяющий все модули через консольное меню.

Инструкция:
чтобы программа работала коректно
все фалы должны быть в одном дериктории и на компьютере должен быть интерпритатор пайтон
